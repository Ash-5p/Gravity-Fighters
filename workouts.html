Powerlifting Workouts

Powerlifting workouts are centered around the 3 main compound lifts: Bench-Press, Deadlift & Squat. The structure each of the following workouts follow is to perform increasingly heavier sets the corresponding, while simultaneously reducing number of reps. The weight lifted each set is calculated as a percentage of your One-Rep-Max (1RM).

1RM Test
Exercise Number of Sets Number of Reps Rest Between Sets
BP/DL/Sq 1 6-10 2min 30sec
BP/DL/Sq * 3 (Failure) 1min - 5min

*Increase weight each set until only 3 reps are achievable. This is 80% of 1RM

Bench-Press

Olympic Weightlifting Workouts

The Snatch and Clean & Jerk which Olympic Weightlifting is centered around are 2 very technical lifts. Both exercises are compound movements (engage multiple muscle groups at the same time) which require flexibility/mobility, power and timing. The exercises in the following workouts focus on different aspects of each lift which will help prepare the body for the full movement.
